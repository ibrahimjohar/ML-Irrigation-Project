# ML-Irrigation-Project

# Canal Irrigation Prediction (Ebro Basin)

Short: Building an irrigation scheduling model and demo dashboard.

MVP (by Dec 2): pipeline that ingests a small Ebro sample, produces features+labels, trains a baseline XGBoost model, and serves recommendations via Streamlit.

Repo layout:
- data/raw/  (raw downloads, NOT committed)
- data/processed/
- notebooks/
- preprocess/
- features/
- models/
- dashboard/
- reports/
- docs/
